# Project-Midnight
Game Project by french young people

Titre : Project: Midnight
Genre : Indéfini pour le moment
Support : Microsoft Windows
Sortie : Été 2020

Le pitch : Dans un futur proche, des adolescents aux capacités surnaturelles sont internés dans un centre de recherches dirigé par une corporation
mystérieuse. Lily est l'une d'entre eux. Lorsqu'elle découvre n'être qu'un rat de laboratoire pour des expériences, elle fuit avec quelques autres.
Maintenant, ils sont poursuivis par des mercenaires bien décidés à les capturer

Le gameplay : Le gameplay est axé sur les pouvoirs de Lily (télépathie, télékinésie, électrokinésie, téléportation, métamorphose, contrôle de l'envrionnement)
Elle peut crafter des objets (un pistolet en tuyau par exemple) et hacker des systèmes informatiques avec son ordinateur portable. Lily ne sera pas le seul personnage
jouable. Entre deux chapitres, on joue Marc, le mercenaire. Son gameplay s'articule surtout sur du shoot.

Le développement : Le développement devrait prendre de 4 à 5 mois si on s'y met à fond, une pre-alpha sortira d'ici Mai ou Juin

On souhaiterait utiliser Unreal Engine 4 pour créer le Jeu
